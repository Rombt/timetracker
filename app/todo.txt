ТЕХ ЗАДАНИЕ
   ЦРМ система для учета рабочего времени. 
      Необходимо создать небольшую ЦРМ на чистом PHP 
      с применением MVC паттерна. 
      стилизация страниц должна быть в файле CSS.

   Необходимый функционал:
      1. Форма входа и регистрации нового юзера. 
         В базе будут юзеры с правами 
            администратора 
            оператора.
      2. При удачной авторизации юзер попадает в журнал учета рабочего времени. 
            Оператор видит только свои записи. 
            Администратор - все.
      3. В журнале учета времени есть такие поля: 
            Пользователь 
            К-во часов
            День
            Комментарий
      4. Должна быть возможность внести новую запись или редактирования существующей. 
      5. Запретить вносить за один день более одной записи. 
      6. Оператор может вносить время только для себя. 
      7. Администратора - для любого оператора.
      8. Для администратора сделать возможность просмотра всех операторов в базе с возможностью сброса им пароля.
      9. Поля должны валидироваться на стороне backend.
